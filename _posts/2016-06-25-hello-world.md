# 宽代码测试 for mobile：

```php
namespace hello;

class world
{
    public static $helloWorld = 'Hello World';
    
    public static function helloWorld ($helloWorld = 'Hello World')
    {
        return $helloWorld;
    }
}

echo hello\world::helloWorld(self::$helloWorld);
```
